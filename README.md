# ITMO_ScientificPython_2026
## Task 1
To get the images back from the `1_1.txt` and `1_2.txt` files, do the following:
1. clone this repo or make sure that you have the `1_1.txt`, `1_2.txt` and `first.py` files in the same directory
2. to get the images run `python ./first.py to_pic`
3. if you want to get back to the text format, run `python ./first.py to_text`
